# YouTubeReborn
iOS YouTube 增强插件，支持最新版YouTube v17.28.2

# 功能介绍

- 去除广告
- 启用后台播放
- 隐藏导航栏
- 自定义颜色
- iPhone上启用iPad风格
- 禁用YouTube儿童
- 下载视频/音频
- 开启画中画功能
- 允许蜂窝数据播放高清视频
